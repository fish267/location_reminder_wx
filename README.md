## 思考

1. 一个小程序该有什么样的能力?

- 能用, 够用
- 上手简单
- 专一

2. 打算实现的功能

- 站点添加
- 站点维护: 站点删除, 站点距离显示, 颜色过度
- 地图展示模块: 自定义站点添加, 当前行驶路径划线, 地图点标

3. 服务端需不需要

- 一期不考虑服务端, 数据存储使用手机端 localStorage
- 二期会开始玩转树莓派, 到时候开个最弱的 mongodb 接口

4. 数据 API 花费

- 个人开发者身份, 高德 API 免费调用次数有限, 如果真的有量, 可以考虑付费模式
